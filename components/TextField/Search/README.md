# TextFieldSearch

Search text-field component;

## Feat

- Fully set up search field

## Syntax

```vue
<TextFieldSearch v-model="search" />
```

## Mixins

### `TextFieldMixin`

> Contains repeated props for text-fields

## Props

### `label`

- Type: `String`

- Default: `'Search products'`

### `placeholder`

- Type: `String`

- Default: `'Press enter to search'`
