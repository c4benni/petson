# ProductCard

Renders the product cards that are on the main page, and category page.

## Syntax

```vue
<!-- regular -->
<ProductCard v-bind="product" />
```

## Props

### `uuid`

- Type: `String`

- Required: `true`

### `title`

- Type: `String`

- Required: `true`

### `image`

- Type: `String`

- Required: `true`

### `brand-title`

- Type: `String`

- Default: `''`

### `price`

- Type: `Number`

- Required: `true`

### `show-add-to-cart`

- Type: `Boolean`
