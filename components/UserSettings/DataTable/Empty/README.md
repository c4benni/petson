# UserSettingsDataTableEmpty

Displace a loading table to in the `body` slot of the `v-data-table`

## Syntax

```vue
<UserSettingsDataTableEmpty :loading="loading" @on-refresh="getOrders" />
```

## Props

### `loading`

- Type: `Boolean`

## Events

### `on-refresh`

#### Description

Fires when the refresh button is clicked.
