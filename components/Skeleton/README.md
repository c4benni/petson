# Skeleton

Renders a shimmer when its `loading` prop is `true`, and its `<slot />` content if any

## Syntax

```vue
<Skeleton :loading="loading" class="w-[250px] h-[32px]">
    ...
</Skeleton>
```

## Props

### `loading`

- Type: `Boolean`

- Default: `true`

### `loading-class`

- Type: `String`

- Default: `undefined`

### `color`

- Type: `String`

- Default: `'grey-300'`
